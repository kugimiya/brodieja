**Установка.**
1. Файл *config.php* заполните своей информацией (login, host, domain, etc.)
2. Откройте в браузере *inst.php*. Скрипт создаст все необходимые базы данных и таблицы.
3. Удалите файл *inst.php*. Это прямая угроза вашему ресурсу удаления всей информации!
4. Проверьте и измените файл *.htaccess* по своим усмотрениям.
